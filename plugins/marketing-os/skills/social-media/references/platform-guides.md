# Social Media Platform Guides

Quick reference for platform specifications, best practices, and content formats.

## LinkedIn

| Spec | Value |
|---|---|
| Post character limit | 3,000 |
| Optimal post length | 100-300 words |
| Image size (single) | 1200x627 or 1080x1080 |
| Carousel (PDF) | 1080x1080 per page, up to 300 pages |
| Video length | Up to 10 min (optimal: 1-3 min) |
| Hashtag limit | 3-5 recommended |
| Link placement | First comment (better reach) |
| Best posting days | Tuesday-Thursday |
| Best posting times | 8-10am, 12-1pm (local business time) |
| Post frequency | 3-5 per week |

**Content that works:** Personal stories, professional insights, data/research, industry trends, career advice, behind-the-scenes, contrarian takes

## X / Twitter

| Spec | Value |
|---|---|
| Tweet character limit | 280 (optimal: 71-100) |
| Thread length | 3-10 tweets (optimal: 5-7) |
| Image size | 1200x675 (16:9) or 1080x1080 |
| Video length | Up to 2:20 (optimal: 30-60s) |
| Hashtag limit | 1-2 per tweet |
| Best posting days | Weekdays |
| Best posting times | 9am-12pm |
| Post frequency | 1-5 per day |

**Content that works:** Hot takes, quick tips, threads with structured insights, reposts with commentary, polls, lists

## Instagram

| Spec | Value |
|---|---|
| Caption limit | 2,200 characters |
| Optimal caption | 100-200 words |
| Feed image | 1080x1080 (square) or 1080x1350 (4:5 portrait) |
| Carousel | Up to 10 slides, 1080x1080 or 1080x1350 |
| Stories | 1080x1920 (9:16), 15s per slide |
| Reels | 1080x1920, 15-90 seconds (optimal: 30-60s) |
| Hashtag limit | Up to 30 (use 10-20) |
| Best posting days | Tuesday-Friday |
| Best posting times | 11am-1pm |
| Post frequency | 3-5 feed posts/week + daily stories |

**Content that works:** Carousel tips, behind-the-scenes, user-generated content, reels with trending audio, educational content, quotes/motivation

## Facebook

| Spec | Value |
|---|---|
| Post character limit | 63,206 |
| Optimal post length | 40-80 characters |
| Image size | 1200x630 |
| Video length | Up to 240 min (optimal: 1-3 min) |
| Hashtag limit | 1-3 (less important on FB) |
| Best posting days | Wednesday-Friday |
| Best posting times | 12-3pm |
| Post frequency | 3-5 per week |

**Content that works:** Community questions, shareable tips, video content, user stories, events, live video

## TikTok

| Spec | Value |
|---|---|
| Video length | Up to 10 min (optimal: 15-60s) |
| Video format | 1080x1920 (9:16) vertical |
| Caption limit | 2,200 characters |
| Optimal caption | < 150 characters + hashtags |
| Hashtag limit | 3-5 |
| Best posting days | Tuesday-Thursday |
| Best posting times | 7-9am, 12-3pm, 7-11pm |
| Post frequency | 1-3 per day (minimum 3-5/week) |

**Content that works:** Trending audio/formats, educational tips, day-in-the-life, before/after, POV content, behind-the-scenes, tutorials, storytelling
