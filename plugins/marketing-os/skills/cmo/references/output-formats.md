# Marketing OS Output Formats

Standard templates for structuring specialist and CMO outputs.

## Quick Summary

Use for fast answers and status checks.

```
## [Topic] Summary

**Key Finding:** [One-sentence insight]

- [Metric 1]: [Value] ([Change])
- [Metric 2]: [Value] ([Change])
- [Metric 3]: [Value] ([Change])

**Recommendation:** [One-sentence action]
```

## Detailed Report

Use for audits, deep analyses, and strategic recommendations.

```
## [Report Title]

**Brand:** [Name] | **Domain:** [domain] | **Date:** [date] | **Period:** [range]

### Executive Summary
[2-3 sentences: what was analyzed, key finding, top recommendation]

### Key Metrics

| Metric | Current | Previous | Change |
|---|---|---|---|
| [Metric] | [Value] | [Value] | [+/-X%] |

### Findings

#### [Finding 1 Title]
[Description with specific data points]

#### [Finding 2 Title]
[Description with specific data points]

### Opportunities
1. **[High Impact]** [Description] — Est. impact: [metric improvement]
2. **[Medium Impact]** [Description] — Est. impact: [metric improvement]
3. **[Quick Win]** [Description] — Est. impact: [metric improvement]

### Recommendations
1. [Action item with specifics]
2. [Action item with specifics]
3. [Action item with specifics]

### Next Steps
- [ ] [Immediate action]
- [ ] [This week]
- [ ] [This month]
```

## Content Brief

```

**Target Keyword:** [keyword] (Volume: X, KD: X)
**Secondary Keywords:** [kw1], [kw2], [kw3]
**Search Intent:** [informational/commercial/transactional/navigational]
**Word Count Target:** [X words]
**Target Audience:** [Description]

### Outline
1. [H2: Section Title]
   - [Key points to cover]
2. [H2: Section Title]
   - [Key points to cover]

### SEO Requirements
- Title tag: [Suggested title] ([X chars])
- Meta description: [Suggested description] ([X chars])

- External links: [Authoritative sources to reference]

### Competitor Content Analysis
| Competitor | URL | Word Count | Backlinks |
|---|---|---|---|

### Key Differentiator
[What makes our content unique/better]
```
