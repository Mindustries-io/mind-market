# Channel Content Formats

Specifications and best practices for each marketing channel.

## Blog / Website

| Element | Specification |
|---|---|
| Title tag | < 60 characters, include target keyword |
| Meta description | 120-155 characters, include CTA |
| H1 | One per page, matches search intent |
| Word count | 1000-3000 (match top-ranking competitor average) |
| Images | 1 per 300-500 words, with alt text |
| Internal links | 3-5 per post minimum |
| External links | 2-3 authoritative sources |
| Reading level | Grade 8-10 (Flesch-Kincaid) |
| URL slug | Short, keyword-rich, hyphens |

## Email

| Element | Specification |
|---|---|
| Subject line | 30-50 characters, no spam triggers |
| Preview text | 40-90 characters, complements subject |
| Body length | 200-500 words (newsletters: up to 800) |
| CTA buttons | 1 primary, 1 secondary max |
| Images | Keep minimal (many clients block by default) |
| Personalization | Use first name, relevant segments |
| Mobile width | 600px max for good mobile rendering |
| Plain text | Always include plain text version |

## LinkedIn

| Element | Specification |
|---|---|
| Post length | 100-300 words (optimal for engagement) |
| Format | Hook line (< 150 chars) -> line break -> body -> CTA |
| Hashtags | 3-5, mix of broad and niche |
| Images | 1200x627 or 1080x1080 |
| Video | Native upload, < 10 min, subtitles |
| Best posting times | Tue-Thu, 8-10am or 12-1pm (business hours) |
| Tone | Professional but personable |

## X / Twitter

| Element | Specification |
|---|---|
| Tweet length | < 280 characters (optimal: 71-100) |
| Thread format | Hook tweet -> 3-7 value tweets -> CTA tweet |
| Hashtags | 1-2 per tweet max |
| Images | 1200x675 (16:9) or 1080x1080 |
| Video | < 2:20 min, auto-play friendly |
| Best posting times | Weekdays, 9am-12pm |
| Tone | Concise, punchy, conversational |

## Instagram

| Element | Specification |
|---|---|
| Caption length | 100-200 words (max 2200 chars) |
| Format | Hook sentence -> line breaks -> body -> CTA -> hashtags |
| Hashtags | 10-20, mix of sizes (separate from caption or first comment) |
| Feed images | 1080x1080 (square) or 1080x1350 (portrait) |
| Stories | 1080x1920 (9:16) |
| Reels | 1080x1920, 15-90 seconds |
| Best posting times | Tue-Fri, 11am-1pm |
| Tone | Visual-first, authentic, community-oriented |

## Facebook

| Element | Specification |
|---|---|
| Post length | 40-80 characters (short posts perform best) |
| Link posts | Include compelling preview image |
| Images | 1200x630 |
| Video | Native upload, < 3 min optimal, subtitles |
| Best posting times | Wed-Fri, 12-3pm |
| Tone | Conversational, community-focused |

## TikTok

| Element | Specification |
|---|---|
| Video length | 15-60 seconds optimal |
| Format | 1080x1920 (9:16), vertical only |
| Caption | < 150 characters + hashtags |
| Hashtags | 3-5, trending + niche mix |
| Hook | First 3 seconds are critical |
| Tone | Authentic, trend-aware, entertaining |
