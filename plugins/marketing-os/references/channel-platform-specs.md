# Channel & Platform Specifications

Single source of truth for per-channel content formats, sizes, timing, and best practices. Used by content-director (channel adaptation) and social-media (platform posts).

## Blog / Website

| Element | Specification |
|---|---|
| Title tag | < 60 characters, include target keyword |
| Meta description | 120-155 characters, include CTA |
| H1 | One per page, matches search intent |
| Word count | 1000-3000 (match top-ranking competitor average) |
| Images | 1 per 300-500 words, with alt text |
| Internal links | 3-5 per post minimum |
| External links | 2-3 authoritative sources |
| Reading level | Grade 8-10 (Flesch-Kincaid) |
| URL slug | Short, keyword-rich, hyphens |

## Email

| Element | Specification |
|---|---|
| Subject line | 30-50 characters, no spam triggers |
| Preview text | 40-90 characters, complements subject |
| Body length | 200-500 words (newsletters: up to 800) |
| CTA buttons | 1 primary, 1 secondary max |
| Images | Keep minimal (many clients block by default) |
| Personalization | Use first name, relevant segments |
| Mobile width | 600px max for good mobile rendering |
| Plain text | Always include plain text version |

## LinkedIn

| Spec | Value |
|---|---|
| Post character limit | 3,000 |
| Optimal post length | 100-300 words |
| Format | Hook line (< 150 chars) -> line break -> body -> CTA |
| Image size (single) | 1200x627 or 1080x1080 |
| Carousel (PDF) | 1080x1080 per page, up to 300 pages |
| Video length | Up to 10 min (optimal: 1-3 min), subtitles |
| Hashtags | 3-5, mix of broad and niche |
| Link placement | First comment (better reach) |
| Best posting times | Tue-Thu, 8-10am or 12-1pm (local business time) |
| Post frequency | 3-5 per week |
| Tone | Professional but personable |

**Content that works:** personal stories, professional insights, data/research, industry trends, career advice, behind-the-scenes, contrarian takes.

## X / Twitter

| Spec | Value |
|---|---|
| Tweet character limit | 280 (optimal: 71-100) |
| Thread format | Hook tweet -> 3-7 value tweets -> CTA tweet |
| Thread length | 3-10 tweets (optimal: 5-7) |
| Image size | 1200x675 (16:9) or 1080x1080 |
| Video length | Up to 2:20 (optimal: 30-60s), auto-play friendly |
| Hashtags | 1-2 per tweet max |
| Best posting times | Weekdays, 9am-12pm |
| Post frequency | 1-5 per day |
| Tone | Concise, punchy, conversational |

**Content that works:** hot takes, quick tips, structured threads, reposts with commentary, polls, lists.

## Instagram

| Spec | Value |
|---|---|
| Caption limit | 2,200 characters (optimal: 100-200 words) |
| Format | Hook sentence -> line breaks -> body -> CTA -> hashtags |
| Feed image | 1080x1080 (square) or 1080x1350 (4:5 portrait) |
| Carousel | Up to 10 slides, 1080x1080 or 1080x1350 |
| Stories | 1080x1920 (9:16), 15s per slide |
| Reels | 1080x1920, 15-90 seconds (optimal: 30-60s) |
| Hashtags | Up to 30 (use 10-20, mix of sizes; caption end or first comment) |
| Best posting times | Tue-Fri, 11am-1pm |
| Post frequency | 3-5 feed posts/week + daily stories |
| Tone | Visual-first, authentic, community-oriented |

**Content that works:** carousel tips, behind-the-scenes, user-generated content, reels with trending audio, educational content, quotes/motivation. Encourage saves (saves > likes for the algorithm).

## Facebook

| Spec | Value |
|---|---|
| Post character limit | 63,206 (optimal: 40-80 characters) |
| Link posts | Include compelling preview image |
| Image size | 1200x630 |
| Video length | Native upload, optimal 1-3 min, subtitles |
| Hashtags | 1-3 (less important on FB) |
| Best posting times | Wed-Fri, 12-3pm |
| Post frequency | 3-5 per week |
| Tone | Conversational, community-focused |

**Content that works:** community questions, shareable tips, video content, user stories, events, live video.

## TikTok

| Spec | Value |
|---|---|
| Video length | Up to 10 min (optimal: 15-60s) |
| Video format | 1080x1920 (9:16), vertical only |
| Caption | 2,200 char limit (optimal: < 150 chars + hashtags) |
| Hashtags | 3-5, trending + niche mix |
| Hook | First 3 seconds are critical |
| Best posting times | Tue-Thu; 7-9am, 12-3pm, 7-11pm |
| Post frequency | 1-3 per day (minimum 3-5/week) |
| Tone | Authentic, trend-aware, entertaining |

**Content that works:** trending audio/formats, educational tips, day-in-the-life, before/after, POV, behind-the-scenes, tutorials, storytelling.

## Image aspect-ratio cheat sheet

| Placement | Size / ratio |
|---|---|
| Blog header | 16:9 |
| LinkedIn / Facebook / X link image | ~2:1 (1200x627 / 1200x630 / 1200x675) |
| Square feed post | 1:1 (1080x1080) |
| Instagram portrait | 4:5 (1080x1350) |
| Stories / Reels / TikTok | 9:16 (1080x1920) |
