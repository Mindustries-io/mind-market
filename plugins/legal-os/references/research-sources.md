## Search Strategies

### For GDPR questions:
```
WebSearch: "EDPB guidelines {topic}" OR "EDPB opinion {topic}"
WebSearch: "{relevant DPA} {topic} decision enforcement"
WebSearch: "CJEU {topic} GDPR ruling"
```

### For enforcement precedents:
```
WebSearch: "GDPR fine {topic} {year}" site:enforcementtracker.com
WebSearch: "{DPA name} {topic} enforcement decision {year}"
```

### For regulatory updates:
```
WebSearch: "{regulation} {topic} implementation guidance {year}"
WebSearch: "European Commission {regulation} {topic} {year}"
```

### For multi-jurisdictional questions:
```
WebSearch: "{country1} vs {country2} {topic} GDPR implementation"
WebSearch: "EDPB consistency mechanism {topic}"
```
